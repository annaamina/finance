# 💰 Finance Dashboard

A responsive and user-friendly **Finance Dashboard** built using React.js to help users manage their income and expenses with a clean and intuitive interface.

🔗 **Live Demo:** https://finance-sandy-psi.vercel.app/
📂 **GitHub Repository:** https://github.com/annaamina/finance

---

## 📌 Overview

This project is a frontend application that allows users to track financial activity in a simple and organized way.

Users can:

* Add income and expense transactions
* View overall financial balance
* Switch between different sections like dashboard and insights
* Understand their financial data visually

The focus of this project is on **clean UI, component structure, and smooth user experience**.

---

## ✨ Features

### 📊 Dashboard

* Displays total balance, income, and expenses
* Simple layout for quick financial overview

### ➕ Add Transaction

* Single form for both income and expense
* Easy toggle between transaction types
* Clean and minimal input design

### 💳 Transactions

* View list of all transactions
* Clearly shows type (income/expense) and amount
* Structured for readability

### 🧠 Insights Page

* Dedicated section for financial insights
* Helps users understand spending patterns

### 💾 Data Persistence

* Data stored using **localStorage**
* Transactions remain after page refresh

### 📱 Responsive Design

* Works across mobile, tablet, and desktop
* Layout adjusts for better usability

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Vite)
* **Styling:** CSS
* **State Management:** React Hooks (useState)
* **Storage:** LocalStorage
* **Deployment:** Vercel
